Remasterização do Linux Mint 20 focada em PHP


Há muito que tenho vontade de criar uma distribuição Linux voltada para o desenvolvimento web, que já viesse com praticamente tudo que uso no dia a dia: Apache, PHP, MySQL, agora MariaDB, Git, Composer, PHPUnit e cia. Para que quando eu instalasse já estivesse pronto para usar.

Bem, meu conhecimento não foi suficiente para criar uma distribuição, mas apareceu agora uma remasterização que na prática permite coisa parecida. Eu gero uma ISO customizando o sistema como desejo, instalando todos os pacotes que preciso, removendo os que não preciso e instalando os scripts no
/usr/local/bin e aliases no ~/.bashrc

Ainda existe a possibilidade de criar uma ISO para uso pessoal copiando meu home inteiro para a ISO e ela ficará como está meu sistema agora.

Bem, criei uma ISO com muita coisa para programador PHP e também com a intenção de atrair programadores do Windows, para que usem em seu desktop um sistema mais parecido com o do servidor, que geralmente usa Linux e assim evitar conflitos e ainda por cima, usar um sistema mais robusto e menos vulnerável.
