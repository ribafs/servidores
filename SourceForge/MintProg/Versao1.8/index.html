Como usar o Mint Prog para tirar o maior proveito.

Configurações Finais do OpenCart

- Crie o banco de dados chamado "opencart" no mysql, usando o collation utf8mb_unicode_ci
- Importe o script /var/www/html/sql/opencart.sql para o banco drupal
- Remova este arquivo /var/www/html/cms/opencart/index.html (somente conseguirá acesso ao OpenCart após remover este arquivo)
- Agora já pode abrir o OpenCart com login e senha "admin" em
    http://localhost/cms/opencart/index.php
